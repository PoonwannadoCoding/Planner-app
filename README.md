# Sleep Cycle

## Pain Point in my daliy life
* Have less time to sleep but want to have a fresh wake up

## Design
* The way I design is to be friendly to eye sight when using it in the dark (dark theme), easy to use for UI

## Does this app solve my problem
* This app help me to manage the sleeping time during the work load week. I have test it this weekend by sleeping at 5 AM then wake up at 8:00 AM. Which is my total sleeping time is only 3 hours but when I wake up I don't feel sleepy at all.
